## 🚀 Cách Deploy

### Phương pháp 1: CLI (Khuyến nghị)
```bash
cd "vercel deploy"
npm install -g vercel
vercel
vercel --prod
```

### Phương pháp 2: Script tự động
```bash
# Windows
deploy.bat

# Linux/Mac
chmod +x deploy.sh
./deploy.sh
```

### Phương pháp 3: Vercel Dashboard
1. Upload folder "vercel deploy"
2. Configure project
3. Deploy
